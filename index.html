ARCHIVE5RP
Discord Bot

Официальный Discord-бот для интеграции с базой данных читеров archive5rp.ru. Автоматизирует проверку участников, мгновенно обнаруживает нарушителей при входе на сервер и непрерывно мониторит обновления базы для GTA 5 RP серверов. База живёт только благодаря сообществу — каждая новая запись добавляется вручную участниками. Без вашего вклада архив не пополняется.

➕ Добавить на сервер 🌐 archive5rp.ru
